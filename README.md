# QuickTodo - Расширение для быстрого добавления TODO-комментариев

**Автор:** Мусаллямова Камила Денисовна  
**Группа:** М3110

## Описание плагина

QuickTodo - это простое и удобное расширение для Visual Studio Code, которое позволяет быстро добавлять TODO-комментарии в код с сохранением текущего отступа.

## Возможности плагина

### Основные функции:

1. **Быстрое добавление TODO-комментариев**
   - Команда для создания TODO-заметок
   - Всплывающее окно для ввода текста задачи

2. **Автоматическое сохранение форматирования**
   - Автоматически определяет текущий отступ строки
   - Сохраняет одинаковый уровень вложенности для TODO-комментария

3. **Умная вставка**
   - Вставляет комментарий на новой строке
   - Сохраняет структуру и читаемость кода

### Использование:

1. **Добавление задачи:**
    Введите комбинацию: Ctrl + Alt + D
    Введите комментарий в открывшемся окне
    Нажмите enter, чтобы подтвердить и создать быстрый комментарий

2. **Результат:**

   // Старый код:
   ArgumentParser* CreateParser(const char* program_name,size_t, max_arg_len) {
    
   }

   // Обновленный - после добавления TODO:
   ArgumentParser* CreateParser(const char* program_name,size_t, max_arg_len) {
       // TODO: Добавить содержимое функции
   }
